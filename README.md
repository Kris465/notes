# Приложение "Заметки"

Простое приложение для работы с заметками, реализованное по паттерну MVC.

## Структура проекта

- `main.py` - точка входа в приложение
- `controller.py` - контроллер, обрабатывающий действия пользователя
- `model.py` - модель, работающая с данными (хранение в JSON-файле)
- `view.py` - представление, отвечающее за интерфейс
- `note.py` - класс заметки

## Функциональность

- Создание заметок
- Редактирование заметок
- Удаление заметок
- Просмотр всех заметок

## Запуск приложения

```
python main.py
```

## Зависимости

- loguru
- colorama
- win32_setctime

Установить зависимости можно командой:

```
pip install -r requirements.txt
```
